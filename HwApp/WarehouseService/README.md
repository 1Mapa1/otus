# WarehouseService

Каркас по образцу `BillingService` (те же слои и связи проектов). Сюда **самостоятельно** добавляются сущности домена, поля контрактов, методы репозиториев и реализация обработчиков.

Сейчас: маршруты подключены, MediatR зарегистрирован, обработчики и репозитории не реализованы (`NotImplementedException`). Миграции не включены — после появления модели см. `dotnet ef migrations add` в документации EF.
